# Weather-Time-Agent
A simple Streamlit app that lets you check **current weather** and **local time** for any city using the [OpenWeatherMap API]
## 📦 Requirements

- Python 3.10 or 3.11 (⚠️ avoid 3.12 for TensorFlow compatibility if you expand later)
- Streamlit
- Requests
- 🖥️ Usage
Enter a city name (e.g., New York, London, Tokyo).

Choose between:

Weather → shows temperature in °C and °F plus a short description.

Time → shows the current local time in that city.

Click Get Report to see results.
